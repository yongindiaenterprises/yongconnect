## Yong Connect Website

This is the source code for the Yong Connect website.

## Installation

1. Install [Node.js](https://nodejs.org/en/download/)
2. Install [pnpm](https://pnpm.io/)
3. Clone this repository
4. Run `pnpm install`
5. Run `pnpm run dev`

## Updating Browserslist

If you need to update Browserslist (as per the warning), you can run:

```sh
npx update-browserslist-db@latest
```
